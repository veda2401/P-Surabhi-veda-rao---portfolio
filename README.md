# P-Surabhi-veda-rao---portfolio

Hey! This is my personal portfolio website that I made using just HTML and CSS.
No frameworks, no JavaScript — kept it simple.

## What I used
- HTML5
- CSS3
- Google Fonts

## Sections
- **Home** — intro and stats
- **About** — skills and background
- **Projects** — UAV Swarm RL, Wanderlust, Online Banking System, Bunkmate
- **Certificates** — Web Dev, DSA, Deep Learning in Python, Developing LLMs
- **Contact** — form and social links

## Files
```
├── index.html
├── styles.css
├── quantum.jpg
├── datacamp.png
├── llm.png
├── veda_portfolio.jpg
└── README.md
```

## How to run
Open `index.html` in a browser. That's it.

## GitHub Pages
1. Upload files to a GitHub repo
2. Go to Settings → Pages
3. Select `main` branch → Save
4. Done — site goes live!
